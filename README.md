# Getting Started

## requirements

* Java 17
* gradle
* openssl

### make commands

* make docker
* make app
* make stop
* make clean
* make keys

### Init env

* make docker
* make keys

### Start app

* make app

### Stop app

* make stop

### Clean database

* make clean
